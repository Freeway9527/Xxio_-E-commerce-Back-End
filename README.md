```
`npm install dotenv`
```

## Usage

To run the program, in your teminal type in:
```
`mysql -u root -p`
```

Enter your password when you are prompt, if no password just hit enter.

```
`source db/schema.sql`

`quit`

`npm run seed`

`npm start`
```
